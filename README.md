# Assignment-9

This assignment has a backend (NodeJS) and a frontend (ReactJs) application.

# Backend

- The backend is developed using nodejs
- The login API is connected to MongoDB
- The API is used by the user to check if the login credentials are correct

# Frontend

- We have crated a react application with below pages
  - Login
  - Home
  - Jobs
  - Contact
  - About Us
- All the pages are connected to each other with react-router-dom
- I have used react-bootstrap and material ui as my UI library for styling the webpage
